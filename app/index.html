Oper Kit WEB

Базовая универсальная платформа для построения web портала поддержки заказчика, имеющая клиент серверную архитектуру и позволяющая использовать по модульному принципу продукты Oper Kit

  • Работа на ОС Win, Linux
  • Авторизация и идентификация пользователей
  • Интеграция с ERP системами, почтовыми клиентами, интернет магазинами и др.
  • Управление договорами на предоставление доступа к онлайн платформе
  • Хранение, управление и отображение конструкторской и эксплуатационной документации в формате ИЭТР, PDF и др.
  • Управление электронами каталогами запасных частей

  • Удобный интерфейс пользователя

Oper Kit publisher

Серверное программное обеспечение, обеспечивающее преобразование ИЭТР в эксплуатационную документацию в виде публикаций, предназначенных для вывода на печать (PDF) или в виде электронного контента (HTML)

  • Работа на ОС Win, Linux
  • Возможность интеграции со средствами разработки ИЭТР
  • Использование CSS стилей для преобразование
  • Высочайшая скорость преобразования
  • Поддержка SVG, JPG, PNG, CGM

Oper Kit Web Project

Клиент серверное программное обеспечение, предназначенное для обеспечения и организации совместного процесса создания, редактирования и управления содержимым (контентом) эксплуатационной и технической документации на сложные изделия по средствам web портала.

  • Работа на ОС Win, Linux
  • Работа во всех современных браузерах
  • Возможность управления документацией в облаке
  • Возможность организации многопользовательской совместной работы
  • Возможность интеграции со сторонними редакторами документации
  • Возможность сравнения и слияния документов

Oper Kit Web XML Editor

Клиент серверное программное обеспечение, являющееся визуальным (WYSIWYG) XML процессором для создания и поддержки технической документации в формате XML согласно XML Schema через WEB-браузер.

  • Работа на ОС Win, Linux
  • Работа во всех современных браузерах
  • Полноценный Web-интерфейс
  • Интеграция с различными системами управления контентом
  • Возможность организации многопользовательской совместной работы
